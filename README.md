# 🍪 Safe Cookie

Ferramenta avançada de análise de segurança web - cookies, SSL, headers HTTP e vulnerabilidades.

## 🚀 Quick Start

```bash
# Instalar dependências
npm install

# Rodar aplicação
npm start
```

Acesse: **http://localhost:3000**

## 📋 Requisitos

- Node.js 16+
- npm 8+

## 🔧 Scripts

```bash
npm start      # Produção
npm run dev    # Desenvolvimento (nodemon)
npm test       # Testes
npm run lint   # Linter
```
